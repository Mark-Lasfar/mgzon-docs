# Theming System

You don't choose one theme.  
**You choose all of them.**

## Built-in Themes

| Theme Name     | Preview |
|----------------|-------|
| Default        | Clean & Professional |
| Midnight       | Deep Dark |
| Ocean          | Blue vibes |
| Sunset         | Warm tones |
| Forest         | Green nature |
| Candy          | Playful colors |
| Monochrome     | Pure black & white |

## Features

- Real-time theme switching
- Persisted across sessions
- System preference detection
- Custom theme creation (coming soon)

```tsx
useTheme() // → "dark" | "light" | "ocean" | "sunset"
```

> Works perfectly with Tailwind + Shadcn/ui

Live Preview → Try switching themes in the top bar
