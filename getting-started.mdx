# Getting Started

Welcome to **mgzonAdmin Pro** — The most powerful admin dashboard ever built for developers who want perfection.

## Prerequisites

- Node.js 18+
- pnpm (recommended) or npm/yarn
- Git

## Installation

```bash
git clone https://github.com/yourname/nextadmin-pro.git
cd nextadmin-pro
pnpm install
```

## Environment Variables

```env
# .env.local
NEXT_PUBLIC_APP_URL=http://localhost:3000
GRAPHQL_ENDPOINT=https://your-api.com/graphql
NEXTAUTH_SECRET=your-super-secret-key
```

## Run Development Server

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## First Login

Default admin account:
- Email: `admin@.....`
- Password: `admin123`

> Change it immediately from Settings → Security
